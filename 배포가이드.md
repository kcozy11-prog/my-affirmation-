# 🔊 나의 확언 앱 — GitHub Pages 배포 가이드

## 📱 폰에서 https로 접속하면 TTS + 녹음 모두 작동합니다!

---

## 🚀 배포 방법 (5분 소요)

### 1단계: GitHub 저장소 만들기

1. **GitHub.com** 에 로그인합니다
2. 오른쪽 상단 **+ → New repository** 클릭
3. 설정:
   - Repository name: `my-affirmation` (원하는 이름)
   - **Public** 선택 (무료 호스팅을 위해 필수)
   - ✅ Add a README file 체크
4. **Create repository** 클릭

### 2단계: index.html 파일 업로드

1. 만든 저장소 페이지에서 **Add file → Upload files** 클릭
2. 다운로드 받은 **index.html** 파일을 드래그앤드롭
3. 하단의 **Commit changes** 클릭

### 3단계: GitHub Pages 켜기

1. 저장소 페이지에서 **Settings** (⚙️) 탭 클릭
2. 왼쪽 메뉴에서 **Pages** 클릭
3. Source를 **Deploy from a branch** 로 설정
4. Branch를 **main** / **(root)** 로 선택
5. **Save** 클릭

### 4단계: 접속!

1~2분 후 아래 주소로 접속 가능:

```
https://[내GitHub아이디].github.io/my-affirmation/
```

예시: `https://kimlaw.github.io/my-affirmation/`

이 주소를 **폰 Chrome 브라우저**에서 열면 됩니다!
